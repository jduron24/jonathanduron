## Overview

Inmobiliaria Matrix is a workflow-focused real estate platform built to help teams manage contacts, companies, active transactions, documents, notes, and task progress from a single system.

At its core, the platform combines a modern web frontend with a FastAPI backend and Firebase infrastructure to organize both people and company records, attach role-based transaction checklists, and keep teams aligned on what needs to happen next. Instead of relying on scattered spreadsheets, manual follow-ups, and disconnected tools, the system gives brokers, coordinators, and admins a shared operational workspace.

The application supports client and company pipelines, active transaction tracking, document uploads and file organization, notes and history logs, shared calendar coordination, and user queues so teams can move deals forward with better visibility and accountability. By combining structured workflows with cloud-based data storage and authentication, Inmobiliaria Matrix turns day-to-day real estate operations into a more repeatable and manageable process.

## What I worked on

- Built a full-stack real estate operations platform using Next.js on the frontend and FastAPI on the backend.
- Designed client and company management flows to store contact details, relationships, statuses, and transaction context.
- Implemented checklist-driven workflows for buyer and listing processes so each record could move through a defined sequence of tasks.
- Developed active queue functionality to show team members the current task, progress percentage, and latest updates across ongoing work.
- Built document management features including uploads, folders, favorites, downloads, and cloud storage integration.
- Created note and history systems to preserve activity logs and improve visibility into client and company changes over time.
- Integrated Firebase services for authentication, Firestore data storage, and file storage.
- Added protected API patterns and token verification to secure backend access across the platform.
- Implemented calendar integration for listing and creating shared events through Google Calendar.
- Supported operational export workflows through Firebase Functions for CSV generation, asset packaging, and Google Drive delivery.

## Outcome

Inmobiliaria Matrix improves real estate operations by centralizing transaction management, reducing manual coordination, and giving teams a structured way to track progress across contacts and companies.

The platform makes day-to-day execution more reliable by combining workflow checklists, team queues, document handling, and calendar coordination in one system. It creates better transparency for admins and transaction coordinators, reduces the risk of missed steps, and provides a scalable foundation for managing real estate processes in a more organized, cloud-based way.
